## 📊 Project Overview
An interactive dashboard analyzing video game sales across different regions, platforms, and genres. Built using **SQL** for data processing and **Tableau** for visualization.

## 🛠️ Technologies Used
- **SQL** (MySQL) - Data exploration and analysis
- **Tableau** - Interactive dashboard creation
- **Excel/CSV** - Data cleaning

---

## 🎯 Key Features
- Global sales distribution across regions (NA, PAL, JP, Other)
- Top performing platforms analysis
- Genre-wise sales breakdown
- Critic scores correlation with sales
- Interactive filters for deeper insights

- ## 📈 Key Insights
- **North America** leads in sales with 334.6M sales
- **Action**, **Shooter**, **Sports** genre dominates the market
- **Minecraft** appears to be the top-selling title
- Strong correlation between critic scores and sales performance
